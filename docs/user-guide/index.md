# User Guide

cfd-viz provides two interfaces for visualizing structured CFD data:

1. **CLI** — quick inspection from the terminal
2. **Python API** — scriptable plotting for notebooks and automation

Both interfaces read data through [cfd-io](https://github.com/uahypersonics/cfd-io),
so any format cfd-io supports (HDF5, Plot3D, Tecplot, SU2, split binary) works
out of the box.

- [CLI Usage](cli.md)
- [Python API](api-usage.md)
