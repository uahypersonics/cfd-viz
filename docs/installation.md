# Installation

## Requirements

- Python 3.11 or later

## From PyPI

```bash
pip install cfd-viz
```

## From Source

```bash
git clone https://github.com/uahypersonics/cfd-viz.git
cd cfd-viz
pip install -e .
```

## Development Installation

```bash
pip install -e ".[dev]"
```

This installs testing and linting tools (`pytest`, `pytest-cov`, `ruff`).
